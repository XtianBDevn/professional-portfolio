# Christian Bryant - Professional Portfolio

A modern, full-stack engineer portfolio built with Next.js, Three.js, and GSAP.

## Features

- **Animated Three.js Background**: Interactive particle star field
- **GSAP Animations**: Smooth, professional animations throughout
- **Interactive Resume**: View and download as PDF
- **Project Showcase**: Animated project cards highlighting work
- **Technology Stack**: Animated badges of specializations
- **Contact Form**: Animated contact form for inquiries
- **Responsive Design**: Fully responsive across all devices
- **Google Maps Integration**: Location display in footer

## Technologies Used

### Frontend
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Three.js / React Three Fiber
- GSAP (GreenSock Animation Platform)

### Tools & Libraries
- jsPDF - PDF generation
- html2canvas - DOM to canvas conversion
- React Icons - Icon library

## Getting Started

### Prerequisites
- Node.js 20.x or higher
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/XtianBDevn/professional-portfolio.git
cd professional-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
